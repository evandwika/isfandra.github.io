Flappy Bird

Flappy Bird merupakan game 2D sederhana yang saya kembangkan menggunakan bahasa pemrograman Python. Pemain mengontrol seekor burung yang harus melewati rintangan berupa pipa tanpa menyentuhnya. Game ini menuntut ketepatan waktu dan konsentrasi, karena tingkat kesulitan meningkat seiring bertambahnya skor.

Fitur Utama
  • Kontrol sederhana menggunakan satu tombol
  • Sistem skor otomatis
  • Rintangan pipa dengan jarak acak
  • Deteksi tabrakan (collision detection)
  • Game over ketika burung menyentuh pipa atau tanah
Teknologi yang Digunakan
  • Python
  • Library Pygame
Tujuan Pembuatan

Proyek ini dibuat untuk melatih pemahaman dasar dalam pembuatan game, seperti logika permainan, sistem skor, serta pengolahan event dan animasi sederhana.

Snake Game

Snake Game adalah game klasik 2D yang saya buat dengan konsep sederhana namun menantang. Pemain mengendalikan seekor ular yang harus memakan apel untuk mendapatkan poin. Setiap apel yang dimakan akan membuat tubuh ular semakin panjang sehingga meningkatkan tingkat kesulitan permainan.

Fitur Utama
  • Gerakan ular menggunakan tombol arah
  • Sistem skor
  • Makanan (apel) muncul secara acak
  • Game over ketika ular menabrak dinding atau tubuhnya sendiri
  • Tingkat kesulitan meningkat seiring panjang ular
Teknologi yang Digunakan
  • Python
  • Pygame
Tujuan Pembuatan

Game ini dibuat untuk melatih logika pergerakan, pengolahan input keyboard, serta penerapan kondisi game over dan sistem skor.
